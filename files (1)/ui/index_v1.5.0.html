cURL Reference
All endpoints accept and return JSON. Authenticate with a session token or API key.
POST /api/auth/login — Get a session token
curl -X POST \ -H "Content-Type: application/json" \ -d '{"username":"admin","password":"your_password"}' \ http://localhost:5000/api/auth/login # Response contains a token — use as X-Session-Token on all requests # Session expires after 8 hours
POST /api/auth/logout — Invalidate session
curl -X POST \ -H "X-Session-Token: YOUR_TOKEN" \ http://localhost:5000/api/auth/logout
Using an API Key directly (no login required)
# API keys still work — pass as X-API-Key header curl -H "X-API-Key: YOUR_API_KEY" \ http://localhost:5000/api/records
GET /api/records
curl -H "X-Session-Token: YOUR_TOKEN" \ http://localhost:5000/api/records # Pagination curl -H "X-Session-Token: YOUR_TOKEN" \ "http://localhost:5000/api/records?page=1&limit=20" # Filter by field curl -H "X-Session-Token: YOUR_TOKEN" \ "http://localhost:5000/api/records?filter_key=plan&filter_val=pro"
POST /api/records — Create
curl -X POST \ -H "X-Session-Token: YOUR_TOKEN" \ -H "Content-Type: application/json" \ -d '{"name":"Jane","phone":{"mobile":"555-1234","work":"555-5678"}}' \ http://localhost:5000/api/records
PATCH /api/records/:id — Partial update
curl -X PATCH \ -H "X-Session-Token: YOUR_TOKEN" \ -H "Content-Type: application/json" \ -d '{"plan":"enterprise"}' \ http://localhost:5000/api/records/RECORD_ID
DELETE /api/records/:id
curl -X DELETE \ -H "X-Session-Token: YOUR_TOKEN" \ http://localhost:5000/api/records/RECORD_ID
POST /api/admin/users — Create user
curl -X POST \ -H "X-Session-Token: YOUR_TOKEN" \ -H "Content-Type: application/json" \ -d '{"username":"alice","password":"s3cur3pass","role":"writer"}' \ http://localhost:5000/api/admin/users
PUT /api/admin/users/:username/password — Change password
curl -X PUT \ -H "X-Session-Token: YOUR_TOKEN" \ -H "Content-Type: application/json" \ -d '{"password":"newpassword123"}' \ http://localhost:5000/api/admin/users/alice/password
POST /api/admin/keys — Create API key
curl -X POST \ -H "X-Session-Token: YOUR_TOKEN" \ -H "Content-Type: application/json" \ -d '{"label":"my-app","role":"writer"}' \ http://localhost:5000/api/admin/keys
